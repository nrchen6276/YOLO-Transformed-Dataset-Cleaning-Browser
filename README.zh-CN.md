# YOLO 变换数据集清洗浏览器

这是一个面向 YOLO 视觉训练数据集的桌面清洗工具，用于在多来源、变换增强、合并后的图片组和跨数据集候选对象中进行人工复核。它的核心目标是让人工快速比较相关图片，选择应保留的图源或候选项，同步管理标签信息，并生成后续治理流程可读取的结构化结果。

当前最新版本：**V3.0_202606050530**

## 工具能力

- 打开数据清洗流程生成的图片组复核目录。
- 在同一界面中展示同组图片，支持小键盘式布局。
- 人工选择图源后，同组其他图片作为变换图进入输出分支。
- 图片和 YOLO 标签同步移动，避免只移动图片造成标签错位。
- 新增跨库候选复核（Manual Objects Review），用于复核重复或近重复候选对象。
- 以 `group_manifest.json` 作为候选元数据权威来源，并写入 `manual_selection.json` 供后续治理流程读取。
- 提供 Windows exe，用户不需要通过命令行启动 Python。

## V3.0 更新重点

- 新增第一层工作流标签：图源组筛选、跨库候选复核、诊断与设置。
- 新增 Manual Objects Review 页面，用于按原因、组大小和组号浏览跨数据集候选组。
- 支持单保留和多保留复核模式。
- 可写入 `APPROVED`、`SKIP`、`AMBIGUOUS`、`NEEDS_AGENT_CHECK` 选择结果。
- 不移动主库文件，不删除候选副本，只写人工选择结果。
- 保留 V2.2.4 的非常规图源组目录和动态分组支持。

## 安全边界

跨库候选复核是人工选择结果写入工具，不是哈希扫描器，也不是主库治理移动器。它不创建候选组，不删除候选副本，不移动主库图片或标签。

所有数据治理结论仍保持 **PENDING_AUDIT**，不能直接升级为论文、模型或数据质量实证结论。
